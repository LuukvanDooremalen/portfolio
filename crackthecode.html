Workshops-CrackTheCode

CrackTheCode is een project waar ik en 3 medestudenten en workshop moesten ontwiikkelen over coderen in p5js. WIj hebben in dit project het thema boeven gekozen. En bij deze workshop moeten kinderen een aantal puzzles oplossen door de regels code aan te passen en toe te passen. Voor dit project hebben we ook een website ontwikkeld waar onze workshop opstaat.

Als je op de site komt is dit de hoofdpagina. Zoals je ziet hebben we voor een "coole" technische look gekozen. Dit thema hebben we overal proberen te houden zodat het duidelijk is dat elke workshop van de site crackthecode afkomt. Zo hebben we dit ontwerp ook gebruikt voor het workshop boekje van crack the code. Zie de foto hieronder.

Tijdens dit project heb ik me vooral bezig gehouden met het ontwerpen van de website en de workshop zelf. Ook heb ik op sommige momenten zitten meekijken met het coderen. Dit was een heel leuk en interesant project om te doen. En ik had ook een hele fijn groep om dit project mee te doen. We hadden goeie samenwerking en alles ging volgens de goeie planning

Wil je nu meer weten over dit project klik dan op de link hieronder en neem dan eens een kijkje naar de workshop
